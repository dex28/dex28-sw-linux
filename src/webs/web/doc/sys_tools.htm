System

Page shows general information about System and Process Control, such as:

  • Reboot Albatross - If you want to reboot Albatross just click on OK button.
  • Shutdown Albatross - If you want to shutdown Albatross just click on OK button.
  • Reload DSPs - To reload DSPs just click on OK button.
  • Kill Connections - If you click on OK button all connections will be killed.

Password

If you want to change your password type old password then type and retype new one.

  • Old Change Password - Old password field.
  • New Password - New password field.
  • Retype New Password - Confirm new password field.

Network

This page allows to diagnose internet connection.

  • Host IP Address - IP address to ping to.
  • Resolve Names

Update

This page allows you to update software to newer version. Choose the file using Browse button then click on Upload button.
