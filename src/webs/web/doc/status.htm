Applet displays host utilization, DSP utilization, and port status.

Description
 CPU
  white     Host CPU utilization (%).
green   Used memory (MiB).
red   Inbound CIR (kbit/s).
yellow   Outbound CIR (kbit/s).
 DSP
white   DSP #0 utilization (%).
green   DSP #1 utilization (%).
cyan   DSP #2 utilization (%).
blue   DSP #3 utilization (%).
 Ports
    Line is down. No connection to ELU or DTS.
    Line is up. ELU protocol is down.
    Status of the port is OK.
    Phone is in speech. Status is OK.
    The phone is in ringing state. Status is OK.
    Faulty DTS is connected.
