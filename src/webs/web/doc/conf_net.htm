Network Settings

Page shows general information about Network Settings, such as:

  • Identity
    • Host Name - Name of Host (default value at the gray background).
    • Domain Name - Name of Domain (default value at the gray background).
  • Ethernet Interface
    • DHCP Enabled - When checked DHCP is enabled.
    • IP Address - IP address (default value at the gray background).
    • Network Mask - Network mask (default value at the gray background).
    • Broadcast Address - Broadcast address (default value at the gray background).
    • Detault Gateway - Detault gateway (default value at the gray background).
  • DNS
    • Name server 1 - Directory number 1 (default value at the gray background).
    • Name server 2 - Directory number 2 (default value at the gray background).

The Unit Configuration page allows user to:

  • Save
  • Apply
  • Reset (to default value)
