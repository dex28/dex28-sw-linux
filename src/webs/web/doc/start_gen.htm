To access help about the DEX28 Administration System, select the Help button on the right hand side of the page. The help page that appears will be appropriate for the area of the DEX28 Administration System.

The Help tool bar at the top of this page provides buttons to navigate the help system.

 Works exactly like a browser's back button.
 Works exactly like a browser's forward button
 Opens the Help Table of Contents.
 

There is also an edit box and Search button in the Help tool bar. If you type some words into the edit box and click the Search button the Server searches for those words in the Help pages. The Search looks for the first word, then each subsequent word acts as a filter on the results of the search. So if the first word is not found in any document, there will be no search results, even if all subsequent words have matches in every document. A list of possible pages is produced once the search is finished.
