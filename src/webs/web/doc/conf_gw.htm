Port Settings

Page shows and enables port configuration paramaters:

  • Directory Number - Directory number assigned to port.
  • Speech Compression Scheme
    • G.711 A-Law
    • G.711 u-Law
    • G.726 ADPCM16
    • G.726 ADPCM24
    • G.726 ADPCM32
    • G.726 ADPCM48
    • G.729 A
    • G.729 AB
  • Line Echo Canceller (LEC) Settings - LEC length (from 32 to 128ms).
  • Adaptive Jitter Buffer - jitter is the variation in the time between packets arriving, caused by network congestion, timing drift, or route changes. A jitter buffer can be used to handle jitter. Default value is 80 ms.

The Unit Configuration page allows user to:

  • Save (changes),or
  • Reset all fields to default values

User Management

Page shows general information about users such as:

  • Username - Username or user ID
  • Password - Password. You can change it and save by clicking on Save button.
  • Group - Group which user belongs to. You can change it and save by clicking on Save button.
  • Disabled - You can disable user.

Authorizations for 'user'

Page shows authorizatons for selected user.

  • Directory Number - Directory Number
  • Mode - Is authorization for each Directory Number.
    • User - Ordinary user.
    • Admin - Admin authorization (superuser).
    • Monitor - Monitor authorization.
