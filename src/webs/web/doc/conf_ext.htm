Extender Port Configuration

Page shows information about Extender Port Configuration, such as:

  • Port - Local Port number.
  • DN - Remote Directory number or Port Number to be connected to on the Switch Side DEX28. Absolute port value is given as <dot><number> e.g. .5 == port #5 (starting from 1).
  • IP Address - Switch Side DEX28 IP Address
  • TCP Port - Switch Side DEX28 TCP Port
  • Username - Connect to Switch Side DEX28using this username
  • Password - Connect to Switch Side DEX28using this password
  • Mode - Connect to Switch Side DEX28 using this mode
  • A - Autoconnect on startup indicator
  • R - Reconnect on failure (otherwise provide menu)
     
  • TAU D Option - Built-in TAU D association for this DTS port,
    given in the following format:
    - usb: to run TAU D on USB port for this DTS port, or
    - tcp <portnumber> to run TAU D NVT service on TCP port for this DTS port
